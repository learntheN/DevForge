使用帮助

1. 配置数据库连接

填写数据库连接信息，点击"保存连接"。支持 MySQL、PostgreSQL 和 SQLite。

2. 选择要导出的表

连接成功后，系统会自动加载数据库中的所有表。为每个表选择导出模式：

  • 不导出 - 跳过此表
  • 仅结构 - 只导出 CREATE TABLE 语句
  • 结构+数据 - 导出表结构和 INSERT 数据

3. 配置导出选项

根据需要调整导出选项，如是否添加 DROP TABLE、是否禁用外键检查等。

4. 生成 SQL 文件

点击"生成 SQL 文件"按钮，预览生成的 SQL 语句，确认无误后下载到本地。

所有配置会自动保存到服务器，下次打开页面时会自动恢复。
